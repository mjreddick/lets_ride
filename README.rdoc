== Let's Ride

Let's Ride is web carpool app centered around events. Visitors can do simple searches or sign up for a free account to either create or join rides. 

Let's Ride runs on the following technologies:

* Ruby 2.1.3

* Rails 4.1.7

* Bootstrap 3.3.2

* PostgreSQL 9.4.0

* Eventful API

* RSpec and Factory Girl for testing

* Heroku for deployment

* ...


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
